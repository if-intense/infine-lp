infine

とは？

（オーウェイ）は、イタリア・ボローニャで1948年に創業した、70年以上の歴史と伝統のある美容総合メーカーrolland社のオーガニックコスメブランドです。

現在、全世界70カ国以上で美容サロン様向けの製品を開発・製造しています。〝バイオダイナミックファーム〟を自社で運営し、原材料の多くを自社栽培しているのも特徴の一つです。製品のデザインや広告・販売すべてのプロセスを全て自社で展開し、高品質な製品を世界中のユーザーに届けています。

infineとは？

バイオダイナミック成分をベースにしたOWAYラインの
新しいスタイリングとフィニッシュ剤

天然植物のWAX、樹脂成分、軟化成分がベースの、多彩な用途に対応するクリエイティビティあふれるスタイリングとフィニッシュ剤のシステムです。
保護成分や栄養成分が一日中髪をケアし、湿気やダメージから髪を守りながら、スタイリングを長持ちさせます。

PASTES / WAXES / PUTTY / MUDS

NECTARS /DROPS / CLOUD

ヘアケア：栄養を与え内部を再構築し、輝きのある魅力的な髪にします。

GLAZE / SPRAY / POTION

スタイリング：フォルムと質感を作ります。

成分

酸化ストレスからの保護

バイオダイナミック・マルビウム
アンチエイジング効果を持った活性成分（）様々な要因によって、髪の内部に発生する活性酸素の発生を抑え、皮膚の参加や早期老化を防ぎます。

物理的な損害に対して

オーガニック・ブラックキヌア
健康の髪が本来持つ弾力性や柔らかさを与え、しなやかな状態にします。また、髪表面（キューティクル）を保護しながら、輝きとプロテクト効果をもたらします。

髪や肌の乾燥

アムラ
ビタミンC、カルシウム、リンをふんだんに含み、髪を芯から潤いを与えます。高脂肪の含有量が豊富で、軽いテキスチャーでありながら、毛髪の深部まで栄養を与えます。保湿効果も高く、髪に活力を与え、弾力性をアップさせます。

髪や肌の乾燥

フェアトレード・マルラ
ビタミンC、カルシウム、リンをふんだんに含み、髪を芯から潤いを与えます。高脂肪の含有量が豊富で、軽いテキスチャーでありながら、毛髪の深部まで栄養を与えます。保湿効果も高く、髪に活力を与え、弾力性をアップさせます。
